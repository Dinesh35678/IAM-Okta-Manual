# Rule Creation and User Validation Process

## Procedure

1.  Navigate to **Rules** and click on **Create New Rule**.
2.  In the rule conditions, specify: **Department value is equal to
    "Intern"**.
3.  Select the required **actions** for the rule.
4.  **Activate** the rule.
5.  Create a **sample user account**.
6.  Update the user's profile details and set the **Department** field
    to **Intern**.
7.  Navigate to the **Groups** section.
8.  Verify that the user has been **automatically and successfully added
    to the group**.
