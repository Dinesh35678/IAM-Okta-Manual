# Okta IAM Lab -- Group Creation and User Assignment

## Objective

The objective of this exercise is to create a group in the Okta Admin
Console and assign users to that group.

------------------------------------------------------------------------

## Step 1 -- Access Okta Admin Console

1.  Log in to your Okta Admin Console.
2.  Navigate to the groups management section.

Okta Admin Console → Directory → Groups

------------------------------------------------------------------------

## Step 2 -- Create a New Group

1.  Click **Add Group**.
2.  Enter the group details.

Example:

  Field         Example
  ------------- ------------------------
  Group Name    Intern_Group
  Description   Group for Intern Users

3.  Click **Save**.

------------------------------------------------------------------------

## Step 3 -- Assign Users to the Group

1.  Open the **created group**.
2.  Click **Assign People**.
3.  Select the users you want to add to the group.
4.  Click **Save**.

Navigation:

Okta Admin Console → Directory → Groups → Select Group → Assign People

------------------------------------------------------------------------

## Step 4 -- Verify Group Assignment

1.  Navigate to the group you created.
2.  Open the **People** tab inside the group.
3.  Verify that the selected users appear in the group membership list.

------------------------------------------------------------------------

## Expected Outcome

-   A new group is successfully created in Okta.
-   Selected users are assigned to the group.
-   Group membership is visible in the **Groups** section.

------------------------------------------------------------------------

## Learning Outcome

After completing this exercise, you will understand:

-   How to create groups in Okta
-   How to assign users to groups
-   Basic group-based access management in an IAM platform
